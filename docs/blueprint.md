# **App Name**: Aesthetic Web Studio Landing Page

## Core Features:

- Navbar with Smooth Scroll: Sticky navigation bar with links to sections and smooth scroll functionality.
- Hero Section: Enticing hero section with headlines, sub-headlines, and prominent call-to-action buttons to draw users in.
- Services Section with Animated Cards: Visually appealing cards showcasing services, enhanced with subtle animations on hover.
- Dynamic Pricing Plans: Clear and concise pricing plans displayed in a structured and easy-to-understand format.
- Portfolio Showcase with Hover Effects: Interactive portfolio section showcasing projects with visually engaging hover effects.
- Contact Form with Validation: Interactive contact form that has the Name, Email, Business Type, and Message and validates the information upon submission

## Style Guidelines:

- Primary color: Soft peach (#FFDAB9) to convey warmth and sophistication.
- Background color: Light beige (#F5F5DC) for a clean and minimalist aesthetic.
- Accent color: Muted apricot (#FFB380) to provide gentle highlights and points of visual interest.
- Font: 'Poppins' sans-serif for a modern and aesthetic appeal.
- Use FontAwesome for general icons; use elegant and simple stroked icons in pastel shades for Services section.
- Mobile-first, fully responsive layout using clean and structured HTML and CSS.
- Subtle fade-in animations on scroll for each section to create a smooth user experience.
